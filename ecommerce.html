: Developed This web application leverages the power of Angular, a popular front end framework, to create responsive and interactive interface that allows users to browse, search for, and purchase wide range of products.  Seller can create account, log in and can manage add , update, and delete product.  User can create account, log in, access the products page and can view product details page, add products, buy products.  The application displays a list of products with detailed descriptions, images, prices, category, color, name.  User can search products using keywords such as product name, price, category and color.  A virtual shopping cart allows users to add and remove items as they shop. It calculates the total cost, including taxes and shipping fees.  User can proceed to the checkout process, where they provide shipping information and payment details.
